# Monikaspetitions - Online Petition Platform

## Overview
A web application for creating, viewing, searching, and signing petitions built with Spring Boot.

## Features
- View all petitions
- Create new petitions
- Search for petitions by keyword
- Sign petitions with name and email
- Professional navy blue design

## Technologies Used
- Java 17
- Spring Boot 3.4.11
- Maven
- Thymeleaf (templating)

## Running the Application
1. Clone the repository
2. Run `mvn clean package`
3. Run the application
4. Access at `http://localhost:8080`
